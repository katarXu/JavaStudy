## 标识符

`[\w$_]?[\w\d$_]+`



## 常量

0b 二进制

0	八进制

0x  十六进制



数据类型

| 类型    | 范围                |
| ------- | ------------------- |
| boolean | 1bit(ture/false)    |
| byte    | 1byte($-2^8-2^8-1$) |
| char    | 2byte(unicode)      |
| short   | 2byte               |
| int     | 4byte               |
| long    | 8byte               |
| float   | 4byte(1+8+23)       |
| double  | 8byte(1+11+52)      |

- 整数默认int，小数默认double

- boolean不能类型转换

- byte、char、short运算时先转换为int



## 运算符

- +=会进行强制类型转换，如

  ```java
  short a = 1;
  a += 1; //a = (short)(a + 1);
  ```

- `>>`右移，`>>>`无符号右移



## 数组

- 初始化

  int []a = new int[3];
